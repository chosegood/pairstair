## Prerequisites
```
$ brew install node
$ brew install mongodb
$ npm install -g grunt-cli
$ npm install -g bower
$ gem install compass
```

## Updating tools
Cleaning npm
```
$ npm cache clean
```

Updating Grunt:
```
$ npm update -g grunt-cli
```

Updating Bower:
```
$ npm update -g bower
$ bower cache clean
```
